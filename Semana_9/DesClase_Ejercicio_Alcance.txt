Usando una Estrategia de Hashing Abierto (Hashing Separado)
-----------------------------------------------------------

Construir un Hash Table de diferentes tamaños, para guardar Registros cuya entidad tenga los siguientes atributos:

Entidad 

	string dni;	
	string nombre;	
	string apellido;
	int edad;


Se solicita:
- Definir el key y values de cada estudiante.
- Definir el tamaño de la Tabla y una Funcion HAsh propia. 
- Generar un conjunto de registros de entrada.
- Mostrar todos los elementos de la tabla hash.
- Buscar un estudiante y mostrar sus datos.
